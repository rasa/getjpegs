Usage

getjpegs [drive_number [starting_sector]]

For example, to do drive 0 (aka \\.\PhysicalDrive0), type:

getjpegs 0

To start at sector 12345, type:

getjpegs 0 12345

I don't use this program any more, as GetDataBack is a much more complete solution.
