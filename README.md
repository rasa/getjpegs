# getjpegs [![Flattr this][flatter_png]][flatter]

Recover .jpg files directly from disk.

## Usage

````batch
getjpegs [drive_number [starting_sector]]
````

For example, to do drive 0 (aka `\\.\PhysicalDrive0`), type:

````batch
getjpegs 0
````

To start at sector 12345, type:

````batch
getjpegs 0 12345
````
